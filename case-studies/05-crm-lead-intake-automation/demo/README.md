# Demo Artifacts

These files are 100% synthetic.

They are not based on real leads, emails, phone numbers, customers, prospects, Odoo records, Google Sheets, logs, screenshots, or company data.

They exist only to illustrate the design of the CRM lead-intake workflow described in the case study.

| File | Purpose |
|---|---|
| `sample_lead_email.json` | Synthetic Gmail web-form lead email. |
| `sample_parsed_lead.json` | Synthetic parsed lead fields extracted from the email. |
| `sample_sheet_audit_record.json` | Synthetic Google Sheets audit/staging record. |
| `sample_deduplication_result.json` | Synthetic deduplication decision before CRM creation. |
| `sample_odoo_crm_lead_payload.json` | Synthetic Odoo CRM lead payload. |
| `sample_processing_log.json` | Synthetic run-level processing log. |

Do not use these files to claim production lead volume, conversion improvement, time savings, success rate, error reduction, revenue impact, or real operational performance.
