# Artefactos Demo

Estos archivos demo son 100% sintéticos.

No están basados en proveedores, productos, órdenes de compra, facturas, precios, impuestos, registros de Odoo, logs, capturas ni datos reales de empresa.

Existen únicamente para ilustrar el diseño del workflow human-in-the-loop de precarga de órdenes de compra.

| Archivo | Propósito |
|---|---|
| `sample_purchase_input.json` | Input sintético que representa una solicitud de compra o solicitud derivada de una factura. |
| `sample_ai_resolution.json` | Ejemplo sintético de resolución de campos asistida por Codex. |
| `sample_validation_result.json` | Output sintético de dry-run / validación. |
| `sample_odoo_draft_preload.json` | Representación sintética de una precarga draft de orden de compra en Odoo. |
| `sample_human_review_checklist.json` | Checklist sintético para validación humana. |
| `sample_audit_trail.json` | Trazabilidad sintética desde input hasta revisión del draft. |

No usar estos archivos para afirmar volumen productivo, ahorros, tasa de éxito, reducción de errores ni performance operativa real.
