# Demo Artifacts

Spanish version: [README.es.md](README.es.md)

These demo files are 100% synthetic.

They are not based on real suppliers, products, purchase orders, invoices, prices, taxes, Odoo records, logs, screenshots, or company data.

They exist only to illustrate the design of the human-in-the-loop purchase-order preload workflow.

| File | Purpose |
|---|---|
| `sample_purchase_input.json` | Synthetic input representing a purchase request or invoice-derived request. |
| `sample_ai_resolution.json` | Synthetic example of Codex-assisted field resolution. |
| `sample_validation_result.json` | Synthetic dry-run / validation output. |
| `sample_odoo_draft_preload.json` | Synthetic representation of an Odoo draft purchase order preload. |
| `sample_human_review_checklist.json` | Synthetic checklist for human validation. |
| `sample_audit_trail.json` | Synthetic trace from input to draft review. |

Do not use these files to claim production volume, savings, success rate, error reduction, or real operational performance.
