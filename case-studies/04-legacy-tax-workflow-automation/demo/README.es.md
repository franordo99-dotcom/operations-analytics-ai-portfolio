# Artefactos Demo

Estos archivos son 100% sintéticos.

No están basados en proveedores reales, CUITs reales, tax IDs reales, certificados de retención reales, registros reales de Odoo, exports reales de SICORE, logs, capturas ni datos reales de empresa.

Existen únicamente para ilustrar el diseño del workflow asistido ERP-to-legacy.

| Archivo | Qué ilustra |
|---|---|
| `sample_withholding_record.json` | Registro ficticio de retención después de extracción ERP. |
| `sample_fixed_width_preview.txt` | Preview ficticio estilo fixed-width. |
| `sample_control_workbook_summary.json` | Resumen ficticio de workbook de control. |
| `sample_ui_automation_step.json` | Paso ficticio de automatización de UI legacy. |
| `sample_validation_summary.json` | Resumen ficticio de validación. |

No usar estos archivos para afirmar volumen productivo, emisión de certificados, impresión de PDFs, resultado de compliance, ahorros, tasa de éxito ni performance operativa real.
